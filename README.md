# 🌊 Morphometric Analysis of a Watershed (GIS)

This project presents a morphometric analysis of a watershed using GIS tools and Digital Elevation Models (DEM), focusing on hydrological and terrain characteristics in the Amazon region.

## 📌 Objectives
- Delimit the watershed from a defined outlet point  
- Extract drainage network  
- Calculate morphometric parameters  
- Analyze terrain characteristics  

## 📊 Morphometric Parameters
- Drainage Density  
- Stream Order  
- Basin Area  
- Perimeter  
- Compactness Coefficient  
- Form Factor  
- Circularity Ratio  
- Slope  

## 🛠️ Tools & Technologies
- QGIS  
- Digital Elevation Model (DEM)  
- GIS Hydrology Tools  
- Raster and Vector Analysis  

## 📍 Study Area
Pará State, Eastern Amazon, Brazil  

## ⚙️ Methodology
The analysis was conducted using GIS techniques:

- DEM preprocessing (sink filling)  
- Flow direction and accumulation  
- Stream extraction and ordering  
- Watershed delineation  
- Morphometric calculations  
- Terrain analysis (slope and elevation)
